# JSF-JPA CRUD

I used `ServletContextListener` to control when initialize and destroy the Entity Manager

I used configurations for Mysql [Mariadb] and H2 databases

Techonologies:

Tomcat 9  
Maven  
Jpa 2.1  
Hibernate 4.3.6  
Jsf 2.1
